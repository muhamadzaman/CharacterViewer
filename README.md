# characterviewer

A new Flutter project.


##Add Two Flavor
- simpsonsCharacter: ```flutter run -t lib/main_simpsonsCharacter.dart --flavor simpsonsCharacter```

- wireCharacter: ```flutter run -t lib/main_wireCharacter.dart --flavor wireCharacter```
